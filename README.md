# Приложение погоды в городах на основе данных `openweathermap` 

`react v18`

### Project Setup

```sh
npm install
```

### Компиляция для разработки

```sh
npm run start
```

### Компиляция и билд для production

```sh
npm run build
```

### Автофикс проекта по конфигурации eslint

```sh
npm run lint:ts:fix
```

### Автофикс стилей по конфигурации stylelint

```sh
npm lint:scss:fix
```
